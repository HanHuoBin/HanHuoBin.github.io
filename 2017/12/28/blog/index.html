Mac搭建基于Github的Hexo博客笔记

环境配置

1.安装node.js

官网下载node.js，并安装

2.安装Git

这个网上很多方法

安装Hexo

1.初始化

此时浏览器输入

_config.yml
db.json
node_modules
package.json
scaffolds
source
themes

$ vim _config.yml

deploy:
type: git
repository:
https://github.com/HanHuoBin/HanHuoBin.github.io.git
branch: master

$ hexo generate 或者：hexo g

ERROR Local hexo not found in ~/blog
ERROR Try runing: ‘npm install hexo –save’

npm install hexo –save

$ hexo deploy 或者：hexo d

$ git clone https://github.com/iissnan/hexo-theme-next themes/next

Extensions

Plugins: https://hexo.io/plugins/

Themes: https://hexo.io/themes/

theme: next

$ hexo clean //清除缓存文件 (db.json) 和已生成的静态文件 (public)
$ hexo g //生成缓存和静态文件
$ hexo d //重新部署到服务器
```
